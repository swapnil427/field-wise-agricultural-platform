ðŸ”§ 6. SETUP INSTRUCTIONS
================================================================================

PREREQUISITES:
- Node.js 20+ LTS
- npm 10+ or pnpm 8+
- Git 2.40+
- Modern browser (Chrome 90+, Firefox 88+, Safari 14+)

STEP 1: CLONE REPOSITORY
git clone https://github.com/your-username/field-wise-help.git
cd field-wise-help-main

STEP 2: INSTALL DEPENDENCIES
npm install
# Expected: ~65 packages, ~250MB total

STEP 3: ENVIRONMENT CONFIGURATION
Create .env file with:

# Server Configuration
PORT=8081
HOST=0.0.0.0
CLIENT_URL=http://localhost:5173

# API Keys (REPLACE WITH REAL KEYS)
GROQ_API_KEY=gsk_your_real_groq_api_key_here
VITE_OPENWEATHER_API_KEY=your_real_openweather_api_key_here

# Socket Configuration
VITE_SOCKET_URL=http://localhost:8081

STEP 4: GET API KEYS

GROQ AI API KEY (Free tier available):
1. Visit: https://console.groq.com/
2. Sign up â†’ API Keys â†’ Create API Key
3. Copy key (starts with gsk_)

OPENWEATHER API KEY (Free: 60 calls/minute):
1. Visit: https://openweathermap.org/api
2. Sign up â†’ API keys â†’ Copy default key

STEP 5: START DEVELOPMENT SERVERS
npm run dev:full
# Or individually:
# npm run dev     (Frontend - Port 5173)
# npm run server  (Backend - Port 8081)

STEP 6: VERIFY INSTALLATION
# Check backend health
curl http://localhost:8081/health
# Expected: {"ok":true}

# Check frontend
curl http://localhost:5173
# Expected: HTML response
